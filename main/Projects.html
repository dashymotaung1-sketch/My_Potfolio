Python Tetris Game

A desktop implementation of the classic Tetris game built using Python. Blocks fall from the top and must be arranged to form complete lines.

Purpose: Practice OOP, game logic, and real-time updates.

Challenges:

  • Handling block rotation logic
  • Preventing collisions and overlaps
  • Managing game state in real time

Solution:

  • Stored each Tetromino as multiple rotation states
  • Validated each move before applying it
