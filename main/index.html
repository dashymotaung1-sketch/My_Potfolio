(python) Tetris

This project is a desktop implementation of the classic game Tetris, built using Python. The game recreates the traditional Tetris experience where different shaped blocks (Tetrominoes) fall from the top of the screen, and the player must rotate and position them to form complete horizontal lines

Purpose of the Project The goal of this project was to practice game development fundamentals in Python, strengthen understanding of OOP principles, and implement real-time logic such as collision detection and state management.

Challenge Rotating Tetrominoes is tricky because: Each shape rotates differently. Rotating near walls can cause the piece to go out of bounds. Some rotations overlap existing blocks.

Overcome It Store each Tetromino as multiple rotation states When rotating: Move to the next rotation index. Check if the rotation is valid. If invalid, cancel the rotation.

Java Chess Engine

"Developed a high-performance chess engine from scratch using Java. This project involved implementing the Minimax algorithm with Alpha-Beta pruning to optimize move searching. I focused heavily on memory management and bitboard representation to ensure the engine could evaluate thousands of positions per second. It demonstrates my ability to translate complex game theory into efficient, object-oriented code."

Challenge: How do you search through millions of possibilities without the computer freezing or running out of memory?

Challenge: Ensuring the engine doesn't crash or make an illegal move during these rare moments.

Overcome It: This is where your dedication and detail-orientation come in. You have to write "Unit Tests" for every possible scenario to ensure the engine follows the rules 100% of the time.

Overcome It: Implementing Alpha-Beta Pruning. This "prunes" branches of the logic tree that are clearly bad, allowing the engine to look deeper into the "good" moves without wasting CPU cycles.
